# Simunot

Application React avec TypeScript, Vite, React Router et Zustand.

## Requirements

- **Node.js** : version 16.x ou supérieure
- **npm** : version 7.x ou supérieure

## Installation

Clonez le repository et installez les dépendances avec le flag `--legacy-per-deps` :

```bash
npm install --legacy-per-deps
```

## Développement

Lancez le serveur de développement Vite :

```bash
npm run dev
```

L'application sera accessible à `http://localhost:5173`


## Linting

Vérifiez la qualité du code :

```bash
npm run lint
```

## Déploiement

Déployez l'application sur GitHub Pages :

```bash
git checkout main
npm run deploy
```


L'application sera accessible à : `https://vincentcmlejeune.github.io/simunot`
